IF YOU GET ERRORS:
-----------------
- "npm is not recognized": Node.js not installed
  - Download from: https://nodejs.org
  - Install it, then try again

- "Port already in use": Something is using port 5173
  - Close any other terminal windows
  - Try again

- "Cannot find package.json": Wrong folder
  - Make sure you're in: C:\Users\Karthik's PC\guardian-flow
  - Type: cd "C:\Users\Karthik's PC\guardian-flow"
  - Then: npm run dev
