### 1. Foreign Key Count Verification
Query:
```sql
SELECT
  con.conname AS constraint_name,
  cl.relname AS source_table,
  att.attname AS source_column,
  cl2.relname AS target_table,
  att2.attname AS target_column
FROM pg_constraint con
JOIN pg_class cl ON con.conrelid = cl.oid
JOIN pg_class cl2 ON con.confrelid = cl2.oid
JOIN unnest(con.conkey) WITH ORDINALITY AS src(col, ord) ON true
JOIN unnest(con.confkey) WITH ORDINALITY AS tgt(col, ord) ON tgt.ord = src.ord
JOIN pg_attribute att ON att.attrelid = cl.oid AND att.attnum = src.col
JOIN pg_attribute att2 ON att2.attrelid = cl2.oid AND att2.attnum = tgt.col
WHERE con.contype = 'f'
  AND cl.relname = 'user_roles'
  AND ns.nspname = 'public';
```

Result: 3 foreign keys found
- user_roles_user_id_fkey (user_roles.user_id → profiles.id)
- user_roles_granted_by_fkey (user_roles.granted_by → profiles.id)
- user_roles_tenant_id_fkey (user_roles.tenant_id → tenants.id)

Status: ✅ Both profile relationships are valid and needed
